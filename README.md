# Creating an ML Project - End to End

## STEP 1 :
#### Activating environment
```
conda activate
```

## STEP 2 :
#### Creating an Environment
```
conda create -p venv python==3.12.7 -y
```
## STEP 3 :
#### Now activate environment
```
conda activate venv/
```
## STEP 4 :
#### Create .gitignore file from git and pull into the vscode terminal
```
git pull
```
